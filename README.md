# Requirements

* PHP >= 7.4.0
* PHPUnit >= 9.5.0

# Installation

Install the package through [Composer](https://getcomposer.org/).

Run the Composer require command from the Terminal:

```bash
git clone git@github.com:imajinyun/leetcode-php.git
cd leetcode-php
composer install
```

# Usage

```bash
phpunit -vvv
```
